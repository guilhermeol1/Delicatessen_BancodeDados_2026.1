# Delicatessen_BancodeDados_2026.1
Projeto de Banco de Dados — modelagem conceitual (ER), projeto lógico normalizado até a 3FN e consultas em SQL e Álgebra Relacional sobre o minimundo de uma delicatessen. Desenvolvido na disciplina de Banco de Dados da UFRRJ em um grupo de 5 integrantes.
